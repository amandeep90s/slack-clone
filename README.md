# Slack Clone

A full-featured real-time messaging application built with Next.js, inspired by Slack.

## Key Features

- 📡 Real-time communication
- 👍 Reactions to messages
- 🧵 Threads / Replies system
- ✏️ Editing messages
- 🗑️ Deleting messages
- 🔐 Role-based access control
- 🖼️ Image attachments
- 🔒 Authentication with Next Auth v5
- 📺 Channel creation
- 🏢 Workspace creation
- ✉️ Invite system / Invite codes
- 💬 Direct messaging
- 👥 User profiles

## Tech Stack

- ⚛️ [Next.js 16](https://nextjs.org) — React framework
- 🎨 [Shadcn UI](https://ui.shadcn.com) & [Tailwind CSS](https://tailwindcss.com) — Styling
- 🚀 Deployed on [Vercel](https://vercel.com)

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
